# TeamCardSignal：F7 设置窗口增强计划

## 目标
- 将 `F7` 改为呼出/关闭设置小窗口。
- 在窗口内支持三项配置：
  1. 启用/关闭标记功能
  2. 切换渲染模式（小图标 / 完整卡牌）
  3. 设置标注快捷键
- 接收端渲染完全按本地设置执行（不依赖发送端模式）。

## 方案
1. 扩展设置模型 `TeamCardSignalSettings`：
   - 新增 `PreviewMode`（`IconOnly` / `FullCard`）
2. 新增设置窗口 UI 状态模块（运行时 Panel + Label）：
   - `F7` 打开/关闭
   - `↑/↓` 选择项，`Enter` 或 `←/→` 执行
   - 快捷键项进入“按任意键设置，Esc 取消”状态
3. 输入处理改造：
   - 移除旧 `F7` 直接开关和 `F8` 改键路径
   - 设置窗口打开时，优先消费按键，不触发发送
4. 渲染逻辑改造：
   - `IconOnly`：仅显示中央卡牌图标（原始小图标模式）
   - `FullCard`：使用当前“中央完整卡牌 + 左上角说明”模式
   - 网络接收始终读取本地 `PreviewMode` 决定渲染

## 验收
- `F7` 稳定呼出/关闭设置窗口。
- 三项配置可操作、可保存、重启后生效。
- 双端模式不同情况下，接收端按自身模式渲染。
