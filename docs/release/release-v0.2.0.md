# TeamCardSignal v0.2.0

## 中文

### 亮点
- 新增 `F7` 设置小窗口（战斗内可随时呼出）
- 设置项支持：
  1. 启用/关闭标记功能
  2. 切换显示模式（小图标 / 完整卡牌）
  3. 自定义标注快捷键
- 接收端渲染按本地设置执行，不受发送端模式影响

### 行为说明
- 默认标注键：`B`
- `F7`：打开/关闭设置窗口
- 已移除旧版 `F8` 改键流程

### 安装
1. 下载 Release 附件
2. 将 `TeamCardSignal` Mod 文件放入游戏 `mods/TeamCardSignal/` 目录
3. 启动游戏验证

---

## English

### Highlights
- Added `F7` in-combat settings popup.
- Popup supports:
  1. Enable/disable signaling
  2. Render mode switch (`IconOnly` / `FullCard`)
  3. Signal hotkey rebinding
- Receiver-side rendering now follows receiver local settings only.

### Behavior
- Default signal key: `B`
- `F7`: open/close settings popup
- Legacy `F8` rebind flow has been removed

### Install
1. Download release assets
2. Place mod files into `mods/TeamCardSignal/`
3. Launch game and verify
